# Raw data

Downloaded source files for this project. Large files (>100 MB) are stored with **Git LFS** — after cloning, run `git lfs pull` to download them.

| Subfolder | Files |
|-----------|--------|
| `dunnhumby/` | Kaggle Complete Journey CSVs |
| `statcan/` | `statcan_retail_trade_food.csv`, `statcan_food_cpi.csv` |
| `openfoodfacts/` | `en.openfoodfacts.org.products.csv.gz` |
| `canada_food_waste_reports/` | Government PDF reports |

See the main [README](../../README.md) for download links and licenses.
